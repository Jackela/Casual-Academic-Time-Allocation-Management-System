This folder contains historical, one-off process/test reports moved out of repository root
to keep the root tree focused on long-lived project artifacts.

Archival policy:
- Keep operational and retrospective reports under `docs/archive/<YYYY-MM>/...`.
- Do not place temporary process reports in repository root.
- CI hygiene gate blocks reintroducing these files at root.
