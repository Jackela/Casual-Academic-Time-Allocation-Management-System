# Ralph Progress Log
Started: Wed Feb 19 21:00:00 CST 2026

## Codebase Patterns
- Entity equality: Use id-based equality with null-id fallback to identity (System.identityHashCode)
- All entities use `Objects.equals()` and `Objects.hash()` for null-safety in equals/hashCode
- New entities (no ID) are never equal to each other - equality only works after persistence
- Pattern: Use enums (ApprovalStatus, UserRole) for all status/role values instead of string literals
- Pattern: Path constants should be grouped in Set<String> for cleaner membership checks
